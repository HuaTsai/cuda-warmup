# cuda-warmup

CUDA practice project. The environment is managed by [pixi](https://pixi.sh)
(CUDA 12.9, CMake, Ninja, GoogleTest, clangd).

## Usage

```sh
pixi run test    # configure + build + run tests
pixi run build   # build only
```

## Layout

- `include/` — public headers (host-safe interfaces; includable from plain C++)
- `src/` — CUDA kernels and launchers
- `tests/` — GoogleTest tests

## Setup clangd

clangd is provided by pixi (`clang-tools` + `clang`; the latter ships the compiler
builtin headers and cannot be omitted). Parsing flags for `.cu` files live in
`.clangd` (the CUDA path is repo-relative; `--gcc-install-dir` and
`--cuda-gpu-arch=sm_89` depend on the local system gcc 13 and GPU — adjust them
when moving to another machine). `compile_commands.json` is generated by CMake,
so run `pixi run build` once.
